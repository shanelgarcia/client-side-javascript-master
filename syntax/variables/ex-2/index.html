Best Practice

  • javascript variables should be self-descriptive.
  • javascript names start with a letter.
  • javascript variables do not contain spaces
  • do not use the following operators in your variable names ( = + - * / )
  • there is no limit on the lenght of a variable name
  • do not use words reserved for the javascript language
  • variables names are case sensitive
  • use camelCase when your have more than one word in a variable name
